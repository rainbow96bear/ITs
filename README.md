# ITs
‘it’s 놀이터’로 해석하면 ‘이것은 놀이터’라는 의미와 함께 IT의 놀이터라는 중의적인 의미를 가지고 있습니다.

놀이터는 어린 아이들이 누구나 즐겁게 놀러와서 친구를 사귀는 공간입니다.
IT업계 사람들 누구나 쉽게 놀러오고 친구를 사귀는 등 인맥을 형성할 수 있길 바라는 마음에 ‘IT’s 놀이터’라고 이름을 지었습니다.
